# Hexachloroethane Degradation Analysis
#### Video Demo: <URL HERE>

#### Description:

This project re-analyzes publicly available experimental data from a 
2025 study by Kim and Sedlak at UC Berkeley, examining the temperature-
dependent degradation of hexachloroethane (HCA) using persulfate-based 
in situ chemical oxidation (ISCO).

**Data source:** Kim, Tae-Kyoung; Sedlak, David L. (2025). 
https://doi.org/10.5061/dryad.wdbrv160s
